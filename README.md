# Tamil Nadu Traffic Management System

A comprehensive traffic and transportation management system specifically designed for Tamil Nadu, providing real-time route planning, location tracking, and intelligent transportation solutions.

## Features

- Real-time route management and planning
- Multilingual support (Tamil and English)
- Parking space finder
- EV charging station locator
- Emergency response system
- Carpooling community
- Extensive coverage of Tamil Nadu locations

## Tech Stack

- Python 3.11
- Streamlit
- Folium for maps
- Streamlit-folium for map integration

## Installation

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/tamil-nadu-traffic-management.git
cd tamil-nadu-traffic-management
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run main.py
```

## Project Structure

- `main.py`: Main application entry point
- `pages/`: Individual pages for different features
- `utils/`: Utility functions and helpers
- `data/`: Mock data and data management
- `assets/`: Static assets and styles

## Usage

The application will be available at `http://localhost:5000` after starting the server.
